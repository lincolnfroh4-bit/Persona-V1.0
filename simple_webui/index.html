Persona-builder training

This flow now supports two data layers: pure target vocals to generally learn the voice, plus paired songs where de-personafied guide vocals map into the real target performance. The package now focuses on the direct regenerator and pronunciation/rebuild assets used by Persona v1.0.

Layer 1 Base identity library

30 to 60 minutes of clean target vocals with no guide pairing required.

Layer 2 Paired supervision songs

Target songs plus matching de-personafied guides at the same lyrics and pacing.

Outcome Accurate rebuild model

Use new external guides later and regenerate them into the target artist.
